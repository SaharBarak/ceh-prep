- `oauth_state_mismatch` — "Your sign-in session expired. Please try again."
- `oauth_code_missing` — "Google didn't return the expected authorization response. Please try again."
- `oauth_token_exchange_failed` — "We couldn't complete the Google sign-in exchange. Please try again."
- `oauth_id_token_invalid` — "We couldn't verify your Google account. Please try again."
- `oauth_email_unverified_google` — "Your Google account email isn't verified. Verify it with Google, then try again."
- `oauth_link_conflict` — "This email is already linked to a different Google account. Contact support if you need help."
- `oauth_link_required` — "This email already has a password account. Please sign in with your password to link Google."  (NEW for OAUTH-06)
- `oauth_rate_limited` — falls back to the existing `rate_limited` message (or add it)
