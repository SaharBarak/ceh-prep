## Audio transcript (whisper)

> Track IPs, phone numbers, and usernames from one CLI. Manual OSINT checks waste time fast. GhostTrack pulls common identity clues into one workflow, instead of opening separate tools for every lookup. GhostTrack is a Python OSINT utility — it gives you three primitives: IP geolocation, phone-number fingerprinting, and username checks across social platforms. Under the hood, it combines external IP data sources, the phonenumbers library, and site checks for usernames. So the workflow is simple — enter an identifier, query the right source, and get a compact investigation result back. This is most useful for security teams, journalists, and anyone validating suspicious contacts. It speeds up first-pass verification before you escalate to deeper forensics. It has over 10,000 stars and supports Linux plus Termux. That makes it easy to test on desktop or Android right away.
